## Description
Detect Objects with YOLO and OpenCV using python

## Requirements
#### Programming Language
* Python3 (pyCharm)
#### Libraries
* OpenCV
* Numpy

## Project Setup
* Clone the project
* Open - **detect.py**
* Choose Model cfg and weight:
  * **YOLOv3-320**
  * **YOLOv3-tiny**
* RUN - **detect.py**
* Web-cam **ON** Automatically
* Detect objects in Web-cam
* To **Quit** - Press **Q-key**
